## var/dynamic

* var : 변수의 타입을 유추해서 대체해준다.

```dart
var name = '고양이';
print(name);
name = 1; // ERROR!
print(name);
```

```dart
var name;
name = '고양이';
print(name);
name = 1;
print(name);
```

* dynamic : Dart의 모든 객체의 기반이 되는 유형이다. 타입 변환이 자유롭다. (대부분의 경우 명시적으로 사용할 필요가 없다.)

```dart
dynamic name = '고양이';
print(name);
name = 1;
print(name);
```

### num

- int와 double은 num에 포함되며 다음과 같이 사용 가능하다.

```dart
  int i = 10; // 정수 10
  double d = 10.0; // 실수 10.0
  
  num a = 10;
  num b = 20.0;
```

- dart는 double타입에 int타입을 대입하는 자동 형변환을 지원하지 않는다

```dart
int a = 10;
double b = a; // 에러
```

- num타입에는 int와 double모두 지원 가능하다

```dart
int a = 10;
double b = 20.0;

num c = a; // OK
c = b;	// OK
```

## 상수 final, const

상수를 선언할 때 final 키워드를 앞에 붙이면 수정되지 않는 상수로 사용할 수 있다.

```dart
final String name = "홍길동";
name = "임꺽정" // ERROR
```

- 타입을 생략하고 다음과 같이 짧게 작성할 수 있다.

```dart
final name = "홍길동";
```

const의 경우 final과 동일하게 사용 가능하다. (값 불변)

final과 const의 차이점 : **final의 경우 Runtime에 값이 지정이 되어도 상관 없지만, const는 Build Time에 값이 지정되어야만 한다.**

```dart
final now = new DateTime.now();
print(now);

const now2 = new DateTime.now(); // ERROR!!
print(now2);
```



## 산술 연산자

다트에서는 다음과 같은 산술 연산자를 지원한다

`assert()` 함수는 계산 결과가 참인지 검사한다.


## 증감 연산자

증감연산자는 1씩 증가시키거나 감소시키는 연산자로 후위연산과 전위 연산이 있다. 

- 후위 연산 : [식]++, [식]--
- 전위 연산 : ++[식], --[식]

```dart
var num = 0;
print(num++); // 나중에 계산하므로 0 출력
// num = 1 이 됨
print(++num)); // 먼저 계산하므로 2 출력
// num = 2가 됨
```

## 타입 검사 (is, is! 키워드)

타입 검사를 하는 키워드

- `is` : 같은 타입이면 true
- `is!` : 다른 타입이면 true

```dart
int a = 10;
if (a is int){
  print('정수');
}

String text = 'hello';
if ( text is! int){
  print('숫자가 아님');
}
```



## 형변환 (as 키워드)

형변환에는 `as`키워드를 사용한다. 다른 타입끼리 변환은 안되고 더 상위 개념으로 변환할 수 있다.

- int와 double은 모두 num을 구현하는 타입이지만 서로는 관계가 없기 때문에 형변환이 불가능하다

```dart
var c = 30.5;
int d = c as int; // 에러
```

- int와 double은 모두 상위 개념인 num으로 현변환할 수 있다.

```dart
dynamic d = 30.5;
num n = d; // as num; 생략 가능
```

## Map

* 맵 선언 및 초기화

  ```dart
  Map animal = {
      'Dog' : '개',
      'Cat' : '고양이',
  };
  print(animal);
  
  Map<String, String> animal2 = {
      'Horse' : '말',
      'Cow' : '소',
  };
  
  print(animal2);
  
  /* RESULT
  {Dog: 개, Cat: 고양이}
  {Horse: 말, Cow: 소}
  */
  ```

* Map 내용 삽입, 수정, 삭제

  ```dart
  Map animal = {
      'Dog' : '개',
      'Cat' : '고양이',
  };
  
  animal.addAll({
      'Bat' : '박쥐',
      'Tiger' : '호랑이',
  });
  // {Dog: 개, Cat: 고양이, Bat: 박쥐, Tiger: 호랑이}
  
  animal['Dog'] = '강아지';
  // {Dog: 강아지, Cat: 고양이, Bat: 박쥐, Tiger: 호랑이}
  
  animal.remove('Bat');
  // {Dog: 강아지, Cat: 고양이, Tiger: 호랑이}
  ```

* Keys & Values

  ```dart
  Map animal = {
      'Dog' : '개',
      'Cat' : '고양이',
  };
  
  print(animal.keys);
  print(animal.keys.toList());
  print(animal.values);
  
  /* Result
  (Dog, Cat)
  [Dog, Cat]
  (개, 고양이)
  */
  ```



## enum 타입

* 선언

  ```dart
  enum Status {
      approved,
      rejected,
      pending,
  }
  ```

* enum과 switch-case

  ```dart
  enum Status {
      approved,
      rejected,
      pending,
  }
  void main() {
      Status status = Status.approved;
  
      switch(status){
          case Status.approved:
              print('승인 상태입니다.');
              break;
  
          case Status.rejected:
              print('거절 상태입니다');
              break;
  
          default:
              print('해당되지 않는 상태입니다');
              break;
      }
  }
  ```

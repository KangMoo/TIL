## 파일 목록 불러오기

- 예시

> ```Javscript
> fs.readdir('파일명', function (error, filelist) { … } );
> ```

## 파일을 비동기적으로 읽기

- 예시

> ```Javscript
> var fs = require('fs');fs.readFile(`파일명`, 'utf-8', function (err, description) { … } );
> ```

> ```Javscript
> fs.writeFile(`파일명`,`내용`,'utf8',function (err)  {… });

## 일반 메뉴 다루기

#### 메뉴 컨트롤

- 도구상자에서 MenuStrip 컨트롤 설정
- 메뉴를 더블클릭하면 이벤트가 생성됨
- 단축키 설정 -> 메뉴 -> 속성 -> Shortcutkeys

## 컨텍스트 메뉴

#### 컨텍스트 메뉴란

- 마우스 우클릭 시 나오는 메뉴

 

#### 컨텍스트 메뉴 설정 및 연결

- ContextMenuStrip     컨트롤 설정 -> 폼 위에 놓기

 

#### 활성화 하려면

1. 도구모음에서 컨텍스트 메뉴 드래그

2. 컨텍스트 메뉴 추가

   ​	2-1. 더블클릭하여 이벤트 추가

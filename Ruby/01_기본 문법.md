## 루비의 기본 문법

- 문장 끝에 세미콜론`;`을 붙이지 않으며, `Tab` 보다는 스페이스 21번을 사용하여 구분한다.

### 주석

```Ruby
# 한 줄 주석

=begin
  Ruby에서 여러줄 주석을 사용할 땐
  begin과 end를 사용한다.
=end
```



### 변수

- String, Integer등의 타입을 선언하지 않고, 필요한 변수에 대입하면, 해당하는 타입으로 자동으로 적용된다.
- 로컬변수, 인스턴스 변수, 상수, 글로벌 변수는 다음과 같이 표현된다.

```ruby
# 로컬 변수 : 소문자와 _를 사용
bellstone

# 인스턴스 변수 : 변수명 앞에 @
@bellstone

# 클래스 변수 : 변수명 앞에 @@
@@bellstone

# 글로벌 변수 : 변수명 앞에 $
$bellstone

# 상수 : 대문자로 작성
BELLSTONE
```

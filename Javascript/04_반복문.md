## 반복문

### for

```javascript
for (let i = 0; i < 10; i++) {
  console.log(i);
}
```

```javascript
const names = ['멍멍이', '야옹이', '멍뭉이'];

for (let i = 0; i < names.length; i++) {
  console.log(names[i]);
}
```

### while

```javascript
let i = 0;
while (i < 10) {
  console.log(i);
  i++;
}
```



### for...of

```javascript
let numbers = [10, 20, 30, 40, 50];
for (let number of numbers) {
  console.log(number);
}
```



### for...in

- 객체를 위한 반복문

```javascript
const doggy = {
  name: '멍멍이',
  sound: '멍멍',
  age: 2
};

for (let key in doggy) {
  console.log(`${key}: ${doggy[key]}`);
}
```
